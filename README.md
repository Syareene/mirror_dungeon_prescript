# mirror_dungeon_prescript
指定した階層のランダムなパックを抽選し司令として出すジェネレーターです。  

サイト: https://syareene.github.io/mirror_dungeon_prescript/  

バグや追加して欲しい機能等あれば上の[Issues](https://github.com/Syareene/mirror_dungeon_prescript/issues/new/choose)からどうぞ。  
息抜きに作成してるので、すぐ実装できるかはわかりませんが参考にさせていただきます。
